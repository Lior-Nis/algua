# Algua brand kit

The Algua identity: the **`Algua` wordmark with the aqua "waterline" on its own
capital A**, and that A extracted as the icon. Minimal and sharp, black + one aqua
accent. Full rationale lives in the design spec:
[`docs/superpowers/specs/2026-06-20-…`](../docs/superpowers/specs/2026-06-20-algua-brand-identity-design.md).

## The one rule

**Black + the aqua water blue, nothing else.** Aqua appears only on the A's
crossbar (the waterline); every other letterform is black or paper. No underline,
no third tone. The icon is the wordmark's own A — never a separately drawn A.

## Assets

| File | What | Use |
|---|---|---|
| `algua-wordmark.svg` / `-dark.svg` | **the logo** — `Algua`, A accented | **primary** — README, docs, inline |
| `algua-glyph.svg` | **the icon** — the A alone | mark on light |
| `algua-glyph-dark.svg` | the icon on black | mark on dark |
| `algua-glyph-mono.svg` | single-ink, knockout waterline | one-color print, stamp, laser |
| `banner-dark.svg` | wide black README header | repo header |
| `favicon.svg` | the icon on paper | browser tab |
| `favicon-16/32.png`, `apple-touch-icon-180.png`, `icon-512.png` | raster icons | favicons, app icon, avatar |
| `tokens.json` | the color palette as design tokens | the single source of truth for color |

## Colors

See `tokens.json`. Black (`#000000`) mark + aqua (`#13C2CE`) waterline only; use
`aqua-deep` (`#0A8E99`) where small aqua-on-light needs WCAG AA contrast.

## Don'ts

No recoloring letterforms aqua · no gradients/shadows/glows · no second accent or
third tone · no waterline-as-rectangle (it's the A's own crossbar recolored) · no
tilting the waterline off horizontal · no underline · no separate/duplicate A
beside the wordmark · no stretching or re-tracking the wordmark.

## Regenerating

The kit is generated by [`build.py`](build.py): both the wordmark and the icon use
the real Space Grotesk outline baked to `<path>` (so the SVGs are font-independent
and survive GitHub's SVG sanitizer); the waterline is the A's own crossbar
recolored. To rebuild:

```bash
# 1. cache the font (OFL, one-time):
curl -sL -o brand/.cache/SpaceGrotesk-var.ttf \
  "https://github.com/google/fonts/raw/main/ofl/spacegrotesk/SpaceGrotesk%5Bwght%5D.ttf"
# 2. generate SVGs + tokens:
uv run --with fonttools python brand/build.py
# 3. rasterize the favicon PNGs (see the cairosvg snippet in build history).
```

The committed SVGs/PNGs are the deliverables — regeneration is only needed when
changing geometry, color, or the wordmark.
