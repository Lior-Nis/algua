# Algua brand kit

The Algua identity: a **wordmark + "waterline A" glyph**, minimal and sharp,
near-monochrome with a single aqua accent. Full rationale and rules live in the
design spec: [`docs/superpowers/specs/2026-05-29-…`](../docs/superpowers/specs/2026-06-20-algua-brand-identity-design.md).

## The one rule

**Aqua appears only on the waterline.** Letterforms — including the wordmark's
capital-A crossbar — are always ink or paper, never aqua. One accent, used once.

## Colors

See `tokens.json`. Aqua (`#13C2CE`) = the waterline only; use `aqua-deep`
(`#0A8E99`) where small aqua-on-light needs WCAG AA contrast.

## Don'ts

No recoloring letterforms aqua · no gradients/shadows/glows · no second accent ·
no rounding the apex · no tilting the waterline off horizontal · no `two-tone`
below 32px (use flat) · no stretching/condensing the wordmark.

## Regenerating

The kit is generated by [`build.py`](build.py): the glyph is hand-built geometry;
the wordmark is the real Space Grotesk Medium outline baked to `<path>` (so the
SVGs are font-independent and survive GitHub's SVG sanitizer). To rebuild:

```bash
# 1. cache the font (OFL, one-time):
curl -sL -o brand/.cache/SpaceGrotesk-var.ttf \
  "https://github.com/google/fonts/raw/main/ofl/spacegrotesk/SpaceGrotesk%5Bwght%5D.ttf"
# 2. generate SVGs + tokens:
uv run --with fonttools python brand/build.py
# 3. rasterize the favicon PNGs (see the cairosvg snippet in build history).
```

The committed SVGs/PNGs are the deliverables — regeneration is only needed when
changing geometry, color, or the wordmark.
